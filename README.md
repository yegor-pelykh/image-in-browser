<h1 id="supported-image-formats">ğŸ–¼ï¸ Supported Image Formats</h1>

**image-in-browser** supports a wide range of image formats for both reading and writing:

**Read / Write Formats:**
- **BMP**
- **GIF (including animated GIFs)**
- **ICO**
- **JPG**
- **PNG (including animated APNG)**
- **TGA**
- **PVR**
- **TIFF**

**Read Only Formats:**
- **PNM (PBM, PGM, PPM)**
- **PSD**
- **WEBP (including animated WEBP)**

**Write Only Format:**
- **CUR**
